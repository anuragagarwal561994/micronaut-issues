# Micronaut Issues

This repository contains sample projects that demonstrate specific issues or features related to the Micronaut framework.

## Project Structure

This is a multi-module Maven project with the following modules:

- [issue-11978](./issue-11978/README.md)
- [issue-11983](./issue-11983/README.md)

## Requirements

- Java 21 (GraalVM)
- Maven 3.8+

## Setup Environment (SDKMAN)

```bash
sdk env
```

## Building the Project

To build all modules:

```bash
mvn clean package
```
